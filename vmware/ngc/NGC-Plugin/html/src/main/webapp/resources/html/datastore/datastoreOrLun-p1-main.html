Select Host Device

Select the host device on which you want to create the Datastore/Lun.

Select Storage Platform

Select the storage on which you want to create the Datastore.

Choose Whether to create Datastore And Choose Storage Type

Please choose a Storage type for the Datastore/Lun that you want to create.

Select Storage Pool

Please select a storage pool to allocate storage resources to the Datastore that you want to create.

Set Storage Resource Details

The system automatically creates a Volume to provide storage space for the Datastore. Please set parameters for the LUN.

Select File System Version

Please select a file system version for the Datastore that you want to create.

Set Datastore name

Please set a name for the Datastore to be created.

Information Summary

Please confirm the information of the Datastore/Lun that you want to create.

Set File System Details

The system automatically creates a File System to provide storage space for the Datastore. Please set parameters for the File System.

Set NFS Share Details

The system automatically creates a NFS share to provide share information for the Datastore. Please set parameters for the NFS share.

Set Datastore Info Details

Please set the information for the Datastore to be created.

Information Summary

Please confirm the information of the Datastore that you want to create.
